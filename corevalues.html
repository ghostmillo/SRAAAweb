Core Values

What are Core Values?

Core Values are deeply ingrained principles that guide a company, organization, or an institution's actions that acts as cultural cornerstones (Lencioni, 2002). SRAAA has a set of core values that makes them stand out among others that needs to be consistently done in order to establish their uniqueness .

1. Dignity

2. Respect

3. Cooperation

4. Unity

5. Love of God and Country
